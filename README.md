# KDjA_Ruhmatoo
## Kasutajaliidese disain ja arendus TLÜ - Rühmatöö HTML ja SASS projekt

### Veebisait "Blogila"
Tegemist on pseudosotsiaalmeedialaadse veebisaidiga, kus avalehelt on võimalik minna inimeste veebiblogidesse, mis on samuti osa veebisaidist. 
Ning kuigi teoorias oleks võimalik lisada blogisid juurde nii palju kui mäluruum lubab, siis selles maketis on kaks näiteblogi, millele saab avalehelt liikuda.
Samuti leidub ka "Meist" leht veebisaidile relevantse info kuvamiseks.
Idee oleks suunatud inimestele, kes igatsevad interneti blogikultuuri, ning sooviks ühest keskkonda, kus tekiks kommuun ja antaks vaba võimalus oma blogisin selles pidada.

[Veebisaidi mockupid](https://app.moqups.com/0ZIjgelO0oBoBC1WTuR79w2TwZHlMalb/view/page/ad64222d5)
Kujutatud on avalehte ja blogilehte nii desktop- kui mobiilvaates (piltidena eksportimine kahjuks pole võimalik keskkonnas limiidi tõttu).
